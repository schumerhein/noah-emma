# Noah & Emma — Testplan volledige cyclus

Doe dit met twee accounts: **jouw account op de Mac** (verkoper/admin) en een
**nieuw testaccount op je telefoon** (koper). App op telefoon:
`http://192.168.1.73:9002` (zelfde wifi).

## A. Registratie & onboarding (telefoon)
- [ ] Nieuw account aanmaken via "Registreer hier" (bijv. test@noahemma.nl)
- [ ] Kind-onboarding doorlopen: geslacht, naam, geboortedatum → klopt de maat-suggestie?
- [ ] Leeftijd ouder dan 12 invullen → verschijnt de rode waarschuwing?

## B. Advertentie plaatsen & moderatie
- [ ] Op telefoon: advertentie plaatsen met foto + AI-model (Noah of Emma)
- [ ] Melding "wordt kort beoordeeld" gezien?
- [ ] Op Mac (jouw account): Profiel → Moderatie (beheer) → staat hij in de wachtrij?
- [ ] Goedkeuren → verschijnt hij daarna in Ontdekken/Zoeken op beide accounts?
- [ ] Nog een advertentie plaatsen en deze **afkeuren** met reden → ziet het
      testaccount de rode "Afgekeurd"-melding op het eigen item?

## C. Kopen: chat, bod, deal
- [ ] Op Mac: open de advertentie van het testaccount → "Stuur bericht" → chat werkt beide kanten op?
- [ ] Doe een **bod** → komt het bod als chatbericht binnen?
- [ ] Op telefoon (verkoper): "Deal afronden" in de chat → systeembericht zichtbaar?
- [ ] Verschijnt de deal bij beiden onder Profiel → Bestellingen?
- [ ] Toont de advertentie nu "Verkocht" (banner + doorgestreepte prijs)?
- [ ] Staat het item met "Verkocht"-badge in zoekresultaten en favorieten?
- [ ] Is de verkoopteller op het profiel van de verkoper met 1 opgehoogd?

## D. Reviews
- [ ] Via Bestellingen: "Schrijf een review" → sterren + tekst → opslaan
- [ ] Verschijnt de review op het verkopersprofiel (tabblad Reviews)?
- [ ] Staat er nu "Je hebt ... beoordeeld ✓" in Bestellingen?

## E. Veiligheid
- [ ] Op Mac: open het verkopersprofiel van het testaccount → ⋮ → **Gebruiker melden** → lukt dat?
- [ ] ⋮ → **Blokkeer gebruiker** → verdwijnen de items van het testaccount uit jouw feed en zoeken?
- [ ] Probeer als geblokkeerde partij een bericht te sturen → nette foutmelding?
- [ ] Deblokkeren via Instellingen → Geblokkeerde gebruikers → alles weer zichtbaar?
- [ ] Meld een advertentie via de vlag op de productpagina

## F. Wachtwoord & gast
- [ ] Uitloggen op telefoon → "Vergeten?" met het testadres → komt de herstelmail aan? → nieuw wachtwoord instellen werkt?
- [ ] "Doorgaan als gast" → swipen kan, maar liken/bieden/chatten vraagt netjes om in te loggen?

## G. Groeifunctie
- [ ] Zet de geboortedatum van een kind een jaar terug → springt de maat automatisch mee?
- [ ] Maak de ingestelde maat bewust te klein → verschijnt de 🌱 groei-melding op het profiel?

Noteer alles wat raar voelt of stuk is — ook kleine dingen. Dan fixen we het in één ronde.
